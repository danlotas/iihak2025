# Ход работы

У нашей командый около нулевой опыт в ML, поэтому делали все мы методом проб и ошибок, тестировали разные модели и способы. Изначально пытались реализовать все путем Natasha + LinearSVC(NatashaSTARIY.ipynb), но поняли что модели такой сложности недостаточно для нашей задачи и перешли на более сложный RuBert. В итоге мы получили рабочий прототип, который в дальнейшем будем допиливать и доробатывать. Большие проблемы возникли с тем чтобы разместить это на веб ресурсе, у нас не было опыта хостинга сайтов, поэтому возникли проблемы с бэкендом. На данный момент смогли реализовать только приложение на Python через streamlit. Запускается через "streamlit run app.py". В дальнейшем продукт будет реализован на веб сервисе.
